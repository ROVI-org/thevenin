Summary#

The Thevenin equivalent circuit model is a common low-fidelity battery model consisting of a single resitor in series with any number of RC pairs, i.e., parallel resistor-capacitor pairs. This Python package contains an API for building and running experiments using Thevenin models.

Version: 0.0.1
